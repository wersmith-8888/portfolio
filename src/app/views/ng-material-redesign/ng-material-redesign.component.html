As a technology organization, the development teams took shortcuts and spent time building designs that were not up to par with industry. The UX team was tasked with delivering a design system and UI library for our developer’s use. Working with the development team, we selected the Material UI library (React). To help the development teams to think in a componetized way, we based the design system off of Brad Frost’s Atomic Design methodology. Once these core systems were selected, we started redesigning the current applications with proper visual design principles applied through the lens of Material Design.

Note: This exercise was built to stretch our design system and see if additions needed to be made. The designs you are about to see have UX flaws. Minimal User Centered Design process was applied to these applications, so there are Information Architecture issues. As the Design Lead, I set a vision for the design library, determined the structure of the design system, and redesigned all of our applications with Material Design principles to verify the robustness of the design library / UI library. Outcomes

By the completion of this project, we had a better inventory of components currently used, and the effectiveness of the Material Redesign. We were able to apply material design and establish a design system which helped create consistency across multiple applications impacting 300+ interal users.
